## 项目结构

本仓库为 Hexo 构建后的静态站点，由 GitHub Pages 自动部署：

```
├── index.html              # 博客首页
├── search.xml              # 搜索索引
├── 2026/                   # 博客文章（按年月归档）
│   └── 03/05/我的第一篇博客/
├── archives/               # 归档页面
├── categories/             # 分类页面
├── tags/                   # 标签页面
├── css/                    # 样式文件
├── js/                     # 脚本文件
└── img/                    # 图片资源
```

## 关于本仓库

此仓库为 **GitHub Pages 部署目录**，存放 Hexo 生成的静态文件。博客源码与文章写作在单独的源仓库中进行，构建完成后通过 `hexo deploy` 推送到此仓库。

如需本地预览，可克隆本仓库后使用任意静态服务器（如 `npx serve .`）或直接打开 `index.html`。
